## 2. 行为变更

1. trio 失败摘要渲染器新增 `--ci-outcome` 入参（`success/failure/cancelled/skipped`）。
2. 失败摘要新增两个总览字段：
   - `ciOutcome`
   - `highRiskStaleCount`
3. 当 `ciOutcome=failure` 且存在高危陈旧项（`stale_high/stale_critical/unknown`）时，自动追加：
   - `:red_circle: STALE HIGH-RISK HIGHLIGHT (Auto)`
   - `PR Auto Prompt (Failure/High-Risk)` 可复制文案块
4. `ai-trio-gate` workflow 把 `steps.gate_trio.outcome` 传给渲染器，确保高危提示是 failure-only 触发。
5. PR 模板补充“高危自动提示粘贴位”，便于把 Step Summary 自动文案带回 PR 对话。
