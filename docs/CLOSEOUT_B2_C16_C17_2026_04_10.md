## 2. 本轮新增/收敛能力

1. Session token 生命周期：
   - 新增 `tokenIssuedAt/tokenExpiresAt`
   - 新增 `SESSION_TOKEN_MAX_AGE_MS`（默认 7 天，最小 60 秒）
2. Session mutation 语义：
   - 返回 `errorCode`（`invalid_token_format | invalid_token | token_expired`）
   - 返回 `tokenState`（issued/expires/remaining/expired）
3. HTTP 状态码语义修正：
   - 无效/过期 token 走 `401`（不再被归一化压成 `400`）
4. Godot CLI 输出体积治理：
   - `world-action` 默认摘要
   - `map-layout` 与 `bootstrap-chain` 默认摘要
   - 保留 `--raw` / `--raw-map-layout` 深度调试入口
