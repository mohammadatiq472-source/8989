## 2. 行为变更

1. `renderGateTrioFailureSummary` 新增 `--pr-auto-prompt-output` 入参：
   - failure + high-risk stale 时输出 PR 自动评论 markdown；
   - 其他场景输出空内容，避免误发评论。
2. `ai-trio-gate` workflow 新增 `Resolve PR auto-comment gate`：
   - 仅 `pull_request` 事件尝试自动评论；
   - 仅当 `gate_trio_pr_auto_prompt.md` 非空才触发。
3. workflow 新增 `Post PR auto comment (failure/high-risk)`：
   - 使用 marker `<!-- ai-trio-gate:auto-comment:failure-high-risk -->` 查找历史评论；
   - 命中则更新，未命中则创建，避免重复刷屏。
4. PR 模板新增自动评论模式注释：
   - 默认 `<!-- ai-trio-auto-comment: auto -->`；
   - 人工覆盖改为 `<!-- ai-trio-auto-comment: manual -->`，workflow 自动跳过评论。

## 4. 风险与边界

1. 本卡不改变 `gate:ai:trio` 判定逻辑，只新增 PR 自动评论分发链。
2. 自动评论依赖 workflow token 写评论权限；权限受限场景会退化为 Step Summary + artifact，不影响 gate 结论。
3. 人工覆盖仅控制“是否自动发评论”，不影响失败摘要生成与 PR 模板手工粘贴流程。
